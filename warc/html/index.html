pywb Wayback Web Recorder

  • /live/
    Fetch a live version of url
  • /record/
    Fetch a live version of url but through warcprox recording proxy, recording all traffic.
  • /replay/
    Replay an archived version of url. If not found, show missing url error.
  • /replay-record/
    Replay an archived version of url, If not found, internally call the /record/ handler to fetch and record a copy.

See https://github.com/ikreymer/pywb-webrecorder for more info and latest updates.
